# YICHE

YICHE (Yet another programming language Inspired by C, Hand-crafted for Educational purposes) is an imperative, ~~statically-typed~~ (not yet) programming language.

I created this project to learn about compilers and implement a full compiler pipeline.

## Goals and directions

- Keep grammar simple and predictable
- Design based on C, but play with different variations

## Features

Currently, the feature set is minimal:

1. types: integers
2. declarations: variables and functions
3. control flow: if and while
4. expressions: arithmetic, relation, logical, assignment
